## What this mod does

Chat ToDo List Fabric Mod is a Minecraft Java client-side mod for players who want to manage tasks without leaving chat. It uses the in-game chat as a practical workspace for to-do lists, progress tracking, and checklist-driven interaction.

Alongside the chat workflow, the mod supports local JSON editing so list content and structure can be adjusted on the client. Version 1.2.0 is built for the Fabric environment and stays focused on straightforward task handling, interactive checklist logic, and editable local data.
